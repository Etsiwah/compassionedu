# CompassionEdu - Quick Reference Guide

## 🎯 Quick Start

### For Admins
1. **Login** → Check dashboard metrics
2. **Daily**: Review announcements, check at-risk students
3. **Weekly**: Monitor attendance rates, process fees
4. **Monthly**: Generate reports, update beneficiary records
5. **Help**: Click "Help" in sidebar for detailed guides

### For Staff
1. **Login** → Check dashboard for announcements
2. **Daily**: Record attendance for your classes
3. **Steps**: Get Student ID → Enter subject → Select status → Submit
4. **Help**: Click "Help" in sidebar for step-by-step guides

### For Students
1. **Login** → View your dashboard
2. **Check**: Grades, attendance, fees, announcements
3. **Update**: Portfolio with new achievements
4. **Help**: Click "Help" in sidebar for complete instructions

## 🔑 Common Tasks

### Admin: Add a New Student
1. Dashboard → Click "Students"
2. Click "Add New Student"
3. Fill in required information
4. Click "Save"

### Admin: Record Fee Payment
1. Dashboard → Click "Fees"
2. Find student
3. Click "Record Payment"
4. Enter amount and details
5. Click "Submit"

### Admin: Upload Results
1. Dashboard → Click "Results"
2. Select term/semester
3. Choose bulk upload or manual entry
4. Enter grades
5. Verify and publish

### Staff: Record Attendance
1. Dashboard → Click "Attendance"
2. Enter Student ID
3. Enter Subject
4. Select Date (defaults to today)
5. Choose Status (Present/Absent/Late)
6. Click "Record Attendance"

### Student: Check Your Grades
1. Login → Click "Results"
2. Select term to view
3. See all subject grades
4. Download report card

### Student: Update Portfolio
1. Login → Click "Portfolio"
2. Click "Add Item"
3. Select item type
4. Fill in details
5. Upload documents/photos
6. Click "Save"

## 🔒 Security Best Practices

1. **Never share your password** with anyone
2. **Log out** when you're done, especially on shared computers
3. **Change your password** regularly (every 3-6 months)
4. **Report suspicious activity** immediately
5. **Keep your email updated** for password recovery

## 💡 Pro Tips

### For Admins
- Export data regularly for backup
- Review activity logs weekly for security
- Use bulk operations for efficiency
- Document important decisions
- Keep user information updated

### For Staff
- Record attendance immediately after roll call
- Keep a list of student IDs handy
- Check announcements every morning
- Submit attendance same day
- Log out after each session

### For Students
- Check portal daily for updates
- Maintain 90%+ attendance
- Update portfolio regularly
- Download and save report cards
- Read all announcements
- Track your academic progress
